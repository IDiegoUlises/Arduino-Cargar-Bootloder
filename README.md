# Arduino Cargar Bootloder

Se utiliza un arduino como ISP para programar otro arduino cargando el circuito de arranque(bootloder)

**Necesita**
* Microcontrolador 328P-PU
* Cristal de 16 MegaHertz
* Resistencia de 10k

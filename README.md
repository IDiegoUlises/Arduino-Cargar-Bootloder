# Arduino Cargar Bootloder

Se utiliza un arduino como ISP para programar el programa de arranque(bootloder) de un microcontrolador

**Requerimientos**
* Microcontrolador 328P-PU
* Cristal de 16 MegaHertz
* Resistencia de 10k
